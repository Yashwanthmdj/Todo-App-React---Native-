# React Native Interview Prep App

A simple React Native app built with Expo for learning and practicing basic UI concepts.

## Features
- Simple app screen
- Expo-based setup
- Ready for further React Native learning and experimentation

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the app:
   ```bash
   npm start
   ```

## Tech Stack
- React Native
- Expo
